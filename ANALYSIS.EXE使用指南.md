# ANALYSIS.EXE 使用指南

## 🎮 快速开始

### 1. 启动摄像头识别

在聊天界面中找到"情绪识别（可选）"区域：

```
📷 情绪识别（可选）
通过摄像头识别你的情绪，让AI更懂你

[🎥 开启摄像头] [📸 拍照识别] [⏹️ 关闭] [🗑️ 清除照片]
```

### 2. 拍摄照片

1. 点击 **"🎥 开启摄像头"** 按钮
2. 授权浏览器访问摄像头权限
3. 调整姿势和位置（确保面部清晰）
4. 点击 **"📸 拍照识别"** 按钮

### 3. 发送消息触发分析

- 输入任意消息（如"你好"、"看看我"）
- 点击"发送"按钮
- 系统自动分析照片内容
- **ANALYSIS.EXE 弹窗自动弹出！**

### 各区域功能

#### 📷 IMAGE DATA
- 显示刚才拍摄的照片
- 应用Matrix风格滤镜效果

#### 😊 USER EMOTION ANALYSIS（用户情绪分析）
**显示条件**: 检测到清晰的人脸

**显示信息**:
- **大表情符号** - 直观表示当前情绪
- **情绪类型** - 英文大写标签（HAPPY/SAD/ANGRY等）
- **置信度** - 分析的可信程度（0-100%）
- **微表情分析** - AI专家的一句话解读

**支持的情绪**:
- 😐 NEUTRAL - 中性平静
- 😊 HAPPY - 开心愉悦
- 😢 SAD - 悲伤沮丧
- 😠 ANGRY - 愤怒生气
- 😮 SURPRISE - 惊讶
- 😨 FEAR - 恐惧焦虑
- 🤢 DISGUST - 厌恶反感

#### 🎯 OBJECT DETECTION（物品识别）
**显示条件**: 检测到可识别的物品

**显示信息**:
- 检测状态指示器
- 精简的物品描述（≤25字符）

**可识别的物品类型**:
- 食物和饮料
- 玩具和玩偶
- 日常用品
- 电子设备
- 植物和动物
- 书籍和文具

#### 📊 EFFECTS（效果状态）
显示识别结果对宠物的影响：
- **MOOD** - 心情值变化（±0到±20）
- **HEALTH** - 健康值变化（±0到±20）

## 🎨 视觉特效说明

### Matrix 黑客风格
- **颜色主题**: 黑底绿字（#00ff41）

- **边框发光**: 霓虹绿色光晕效果
- **扫描线**: 持续滚动的扫描线动画

   - 从小到大缩放

   - 持续时间: 0.3秒

2. **文字发光** (Matrix Glow)
   - 文字周围的绿色光晕
   - 呼吸式的明暗变化
   - 周期: 2秒

3. **扫描线** (Scanline)
   - 从上到下滚动的亮线
   - 模拟CRT显示器效果
   - 周期: 4秒

## 🔐 隐私与安全

### 数据处理方式
- ✅ **本地分析**: 照片仅在浏览器和服务器内存中处理
- ✅ **不落盘**: 不保存到数据库或磁盘
- ✅ **不记录**: 不写入日志文件
- ✅ **即时销毁**: 分析完成后立即释放

### 用户控制权
- ✅ **主动授权**: 需要用户明确点击才能开启摄像头
- ✅ **随时关闭**: 可以随时停用摄像头功能
- ✅ **可选功能**: 完全可选，不影响正常聊天

### 隐私提示
在界面底部始终显示：
```
ℹ️ 隐私保护：照片仅用于即时情绪分析，不会保存到服务器
```

## 🛠️ 故障排除

### 问题1: 摄像头无法开启
**可能原因**:
- 浏览器权限未授予
- 其他应用占用摄像头
- 浏览器不支持（需Chrome/Edge/Firefox等现代浏览器）

**解决方法**:
1. 检查浏览器地址栏的摄像头图标
2. 点击"允许"访问摄像头
3. 关闭其他使用摄像头的应用
4. 刷新页面重试

### 问题2: 弹窗没有显示
**可能原因**:
- 没有拍照就直接发送消息
- 网络延迟导致数据未返回

**解决方法**:
1. 确保先点击"拍照识别"再发送消息
2. 等待几秒让AI分析完成
3. 查看浏览器控制台是否有错误

### 问题3: 情绪识别不准确
**可能原因**:
- 光线不足
- 角度不正
- 表情不清晰

**优化建议**:
- 📸 确保正面面对摄像头
- 💡 在光线充足的环境拍摄
- 😊 保持自然的表情
- 📏 调整距离（约30-50cm）

### 问题4: 物品识别为空
**可能原因**:
- 照片中没有清晰的物品
- 物品过小或过模糊
- 物品类型不在识别范围内

**改进方法**:
- 将物品放在画面中心
- 确保物品占据一定比例
- 使用常见的日常物品
