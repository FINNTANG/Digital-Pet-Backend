### 第1步：环境准备和依赖安装 ⏱️ 15分钟
- [ ] 安装Django REST Framework
- [ ] 安装Simple JWT
- [ ] 安装其他必要依赖
- [ ] 更新项目配置

### 第2步：数据模型设计 ⏱️ 30分钟

- [ ] 添加Signal自动创建Profile
- [ ] 运行数据库迁移

- [ ] UserProfileSerializer（用户资料）
- [ ] RegisterSerializer（用户注册）
- [ ] LoginSerializer（用户登录）
- [ ] ChangePasswordSerializer（修改密码）

### 第4步：API视图实现 ⏱️ 2小时

- [ ] UserViewSet（用户管理）
  - me（当前用户信息）
  - update_me（更新信息）
  - change_password（修改密码）
  - upload_avatar（上传头像）
